# Tiled Flame Game

> A Flutter Flame Game project following [Tyler Codes' tutorial](https://www.youtube.com/@TylerCodes).

> Project Status: WIP for Tutorial, with custom Tweaks. Stuck at Collision Detection via Tiled Objects.
> Versions: Flutter v3.35.7; Dart v3.9.2; Flame v1.33.0
